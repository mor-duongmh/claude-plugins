Invoke the `openspec-explore` skill using the Skill tool.

The skill puts Claude into explore mode: a curious, adaptive thinking partner that asks questions, surfaces multiple directions, and investigates the codebase — but does NOT write code or implement features.

When the user is ready to commit to a change, run `/spec:propose` to generate the full proposal + design + tasks artifacts.
